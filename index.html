Workspace Architecture

Welcome to Aura Reader. This harness transforms reading from a passive process into active data extraction.

1. Seamless File Injection

Drag and drop any DRM-free .epub file directly into this screen window workspace, or click the primary action bar on the top left. The book splits into isolated chapter structures inside local storage memory instantly.

2. Deep Neurobiology Optimizations

Toggle advanced cognitive tools designed around visual attention pathways:

  • Quiet Eye Isolation: Mutes surrounding prose blocks, focusing neural tracking mechanisms explicitly onto a single, sharp target sentence line.
  • Optokinetic Tracking: Generates a smooth horizontal guide pacing layout to stimulate sustained alertness metrics via consistent, fluid eye tracking speeds.

3. Constructing Prompt Packages

Highlight text anywhere in an active chapter block to call up the Context Bubble. Choose an operational modality to instantly construct structured system configurations (encapsulated within clean XML anchors) sent directly to your system clipboard for local or cloud LLM deployment.
