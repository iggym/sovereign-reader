    🔮 Quickstart Instructions

    1. Load Text Structures: Drag and drop any unencrypted .epub file directly anywhere onto this layout window. The parsing engine maps chapters instantaneously inside the sidebar navigation workspace.

    2. Constructing Prompt Packages: Highlight text strings anywhere inside an active chapter block to call up the Context Bubble. Choose an operational modality to instantly construct structured system configurations (encapsulated within clean XML anchors) sent directly to your system clipboard.
