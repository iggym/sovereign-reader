🔮 Quickstart Instructions

1. Load Text Structures: Simply drag and drop any unencrypted .epub book file directly anywhere onto this layout window. The parsing engine maps chapters instantaneously inside the sidebar index panel.

2. Trigger Prompt Packages: Highlight text strings anywhere inside an active text segment. A contextual prompt harness array will materialize immediately above your cursor. Click an operational asset profile to instantly package the selection into structured prompt contexts copied directly to your clipboard.
